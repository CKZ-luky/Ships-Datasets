# Ships-Datasets
Dataset repository for academic research
This repository collects the SeaShips(fog) dataset, which is publicly available on the internet and includes artificially fogged images. When using these datasets for research and development work, researchers and engineers should comply with the usage regulations and copyright requirements of the datasets.

The files are shared via a cloud storage service. You can access them using the following link：Seaships(fog)
